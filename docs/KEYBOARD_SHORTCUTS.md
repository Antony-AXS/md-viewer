# Keyboard Shortcuts

All keyboard shortcuts available in MD Viewer.

---

## Global

| Shortcut | Action |
|----------|--------|
| `Ctrl+P` | Open command palette |
| `Ctrl+K` | Focus file search |
| `Ctrl+B` | Toggle sidebar |
| `Ctrl+T` | Toggle table of contents |
| `Ctrl+Shift+Z` | Toggle zen mode |
| `Escape` | Close search / command palette / presentation mode |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Alt+←` | Navigate back in history |
| `Alt+→` | Navigate forward in history |

## Search

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Move through search results |
| `Enter` | Open selected result |
| `Escape` | Clear search and blur |

## Command Palette

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Move through results |
| `Enter` | Execute selected action |
| `Escape` | Close palette |

## Presentation Mode

| Shortcut | Action |
|----------|--------|
| `→` or `Space` | Next slide |
| `←` | Previous slide |
| `Escape` | Exit presentation |

## Tips

- Most shortcuts use `Cmd` instead of `Ctrl` on macOS
- The command palette (`Ctrl+P`) is the fastest way to do anything — it searches files, headings, and actions in one place
- Zen mode is great for focused reading — hover the top-right corner to reveal the exit button
